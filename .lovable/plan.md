
# خطة التعديلات

## 1. تغيير نص البحث
تعديل نص placeholder من "ابحث عن طبق أو طباخ" إلى "ابحث عن طبق أو مطبخ" في ملف الترجمة.

## 2. البحث باسم المطبخ (الطباخة)
عند كتابة اسم مطبخ في حقل البحث، يتم البحث أيضاً في اسم الطباخة (display_name) بالإضافة للبحث في عنوان ووصف الطبق. هذا يعني أنه إذا كتبت اسم مطبخ مثل "أم أحمد" ستظهر جميع أطباقها.

## 3. جعل صفحة الطباخة مرئية فوراً
عند إنشاء حساب طباخة جديد، يتم تفعيل الصفحة تلقائياً (page_enabled = true) بدلاً من أن تكون معطلة. التعديل على مستوى قاعدة البيانات بتغيير القيمة الافتراضية لعمود page_enabled.

---

## التفاصيل التقنية

### الملفات المتأثرة

| الملف | التعديل |
|-------|---------|
| `src/hooks/useLanguage.tsx` | تغيير نص البحث |
| `src/pages/Index.tsx` | إضافة البحث باسم الطباخة (merchant display_name) في فلتر البحث |
| `src/pages/Browse.tsx` | نفس التعديل على فلتر البحث |
| Migration SQL | تغيير default لعمود page_enabled إلى true |

### منطق البحث الجديد
البحث سيتحقق من:
- عنوان الطبق (title)
- وصف الطبق (description)
- اسم المطبخ/الطباخة (display_name من merchantsInfo)

### تغيير قاعدة البيانات
```sql
ALTER TABLE profiles ALTER COLUMN page_enabled SET DEFAULT true;
```
هذا يجعل أي طباخة جديدة تسجل تكون صفحتها مرئية فوراً.
