
# إعادة تصميم الصفحة الرئيسية

## الفكرة العامة

تحويل الصفحة الرئيسية من صفحة ترحيبية (Hero) منفصلة عن الأطباق، إلى صفحة متكاملة تعرض الشعار بحجم صغير في الأعلى ثم الأطباق والتصنيفات مباشرة تحته، مع قسم تواصل ومعلومات في نهاية الصفحة. جميع الوظائف الحالية ستبقى كما هي بدون أي تغيير.

## التغييرات المطلوبة

### 1. تحديث Header.tsx
- اضافة زر "سجلي الآن كطباخة" في الهيدر للزوار غير المسجلين (بجانب زر تسجيل الدخول)
- ازالة شرط `showCartAndAdmin` بحيث تظهر السلة والأدمن دائما (لأن الأطباق أصبحت في الصفحة الرئيسية)

### 2. إعادة كتابة Index.tsx بالكامل
الهيكل الجديد:

```text
┌─────────────────────────────────┐
│           Header                │
├─────────────────────────────────┤
│  شعار صغير + عنوان المنصة       │
│  (شعار ~120px بدل 500px)        │
├─────────────────────────────────┤
│  شريط البحث                     │
│  فلتر الدول (ScrollArea)        │
│  فلتر التصنيفات (ScrollArea)    │
├─────────────────────────────────┤
│                                 │
│  شبكة الأطباق (2-4 أعمدة)      │
│  (نفس كروت Browse.tsx)          │
│                                 │
├─────────────────────────────────┤
│  قسم التواصل والمعلومات         │
│  - واتساب                       │
│  - انستغرام + فيسبوك            │
│  - الشروط والأحكام              │
│  - حقوق النشر                   │
└─────────────────────────────────┘
```

**التفاصيل:**
- نسخ كامل منطق جلب الأطباق والفلترة من Browse.tsx إلى Index.tsx
- استخدام نفس مكونات CategoryFilter و CountryFilter
- نفس كروت عرض الأطباق مع زر الإضافة للسلة
- الشعار يصبح صغير (~120px) مع العنوان بجانبه أو تحته
- إزالة الأيقونات العائمة (Food Icons) والخلفيات الديكورية
- قسم الفوتر يتضمن: واتساب، انستغرام، فيسبوك، الشروط، حقوق النشر

### 3. صفحة Browse.tsx
- تبقى كما هي بدون تغيير (كصفحة بديلة/مكررة)

## التفاصيل التقنية

### الملفات المتأثرة

| الملف | نوع التغيير |
|-------|-------------|
| `src/pages/Index.tsx` | إعادة كتابة - دمج عرض الأطباق من Browse |
| `src/components/Header.tsx` | تعديل - إضافة زر "سجلي كطباخة" + إظهار السلة دائما |

### Header.tsx - التغييرات
- إزالة شرط `showCartAndAdmin` أو تعديله ليشمل الصفحة الرئيسية
- إضافة زر "سجلي الآن كطباخة" للزوار غير المسجلين يوجه إلى `/auth?type=merchant`
- إبقاء كل الأزرار الحالية (لوحة الأدمن، السلة، تسجيل الدخول، مطبخي، حسابك)

### Index.tsx - الهيكل الجديد
- استيراد وإعادة استخدام: `CategoryFilter`, `CountryFilter`, `useCart`, `Search`, `Plus`, etc.
- نسخ منطق `fetchItems`, `filteredItems`, `handleAddToCart`, `getCurrencySymbol` من Browse.tsx
- شعار صغير في الأعلى مع نص "حوّلي شغفك بالطبخ إلى مشروع حقيقي"
- شريط بحث + فلاتر (sticky تحت الهيدر)
- شبكة كروت الأطباق
- فوتر غني بالمعلومات: واتساب، سوشال ميديا، شروط، حقوق نشر
- إبقاء زر WhatsApp العائم في الزاوية
